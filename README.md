- 下载插件的步骤
    1. 点击 `Marketplace`
    2. 在 `Type / to see options` 处输入想搜索的插件
    3. 选择适合的产检点击 `Install`
- 查看已安装插件的步骤
    1. 点击 `Installed` 便可在中间栏查看已安装的插件</br>![img_23.png](img/img_23.png)
- 禁用或卸载插件的步骤
    1. 利用上方步骤的基础上选中要卸载的插件
    2. 右键该插件</br>![img_24.png](img/img_24.png)
    3. 选择 `Disabled(禁用)` 或者 `Uninstall(卸载)`
